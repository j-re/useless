# My CLI App

This project is a command-line application that processes a CSV file, makes API calls, and logs the responses.

## Installation

To install the necessary dependencies, run the following command:

```
npm install
```

## Build

To compile the TypeScript code to JavaScript, use the following command:

```
npm run build
```

## Usage

To execute the application, run the following command:

```
npm start
```

Make sure to have an `input.csv` file in the root directory of the project, as the application reads from this file.

## Dependencies

This project uses the following dependencies:

- `node-fetch`: A lightweight module that brings `window.fetch` to Node.js.
